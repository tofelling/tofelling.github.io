锻炼

周一

灵活性与平衡目标：增加关节活动范围（尤其髋关节和肩关节），改善平衡能力。
时长：45-60分钟
内容：热身（5-10分钟）：原地踏步或跳跃 + 动态拉伸（腿部摆动、臀部旋转）
灵活性训练（15-20分钟）：颈部拉伸：坐直，头缓慢转向一侧，保持5秒，每侧3次

颈部拉伸：坐直，头缓慢转向一侧，保持5秒，每侧3次 (NHS Flexibility)。
腿筋拉伸：一脚放在瑜伽垫上，身体前倾，保持30秒，每侧2-3次 (BHF Stretching)。
猫牛式：四肢着地，背部交替拱起和下沉，重复10次 (Healthline Flexibility)。

平衡训练（15-20分钟）：
单腿站立：单腿站立10-30秒，换腿，重复3次 (Mayo Clinic Balance)。
树式：单腿站立，另一脚放在大腿内侧，保持10-30秒，每侧3次 (Healthline Balance)。
弹力带侧抬腿：用弹力带固定脚踝，侧抬腿，保持5秒，每侧10次 (NHS Balance)。

放松（5-10分钟）：静态拉伸，重点拉伸腿部、背部和颈部。
胸椎旋转与伸展：四足跪姿进行猫牛式（Cat-Cow）和胸椎扭转，放松背部、增加脊柱活动度 (九个扭转体式 帮你修复脊柱能量_椅子) (史上最全猫牛式解析_体式)。
髋关节拉伸：做弓步压髋、臀部桥式（Hip Bridge）配合双腿交替抬高，拉伸髋屈肌并激活臀肌，保持骨盆中立。注意收紧核心，避免下背部过度前凸。
踝关节活动：站立抬脚尖、脚跟点地，踝关节画圈；或单腿下蹲时脚跟抬起锻炼踝屈伸，减少膝关节压力。
肩部与上肢柔韧：使用弹力带做肩部拉伸，如站姿双手握带向上拉伸胸前、双臂向两侧水平张开后再收回等，扩大肩关节活动范围，避免肩胛位置固定不良。

周二：核心与稳定性

目标：增强核心力量和动作稳定性，支持全身动作。时长：45-60分钟
内容：

热身（5-10分钟）：轻度有氧运动 + 动态拉伸。
稳定性训练（15-20分钟）：
鸟狗式：四肢着地，伸展对侧手臂和腿，保持5秒，每侧10次 (Healthline Lumbar Stabilization)。
平板支撑：保持20-30秒，重复3次 (ACE Core Stability)。
单腿站立哑铃推举：单腿站立，用10kg哑铃推举，3组，每侧10次。

核心训练（15-20分钟）：
仰卧起坐：3组，每组10-15次 (NASM Core Training)。
俄罗斯扭转（弹力带）：坐姿，用弹力带增加阻力，每侧10-15次，3组 (ACE Core Stability)。
抬腿：仰卧，腿抬至90度，3组，每组10-15次 (NASM Core Training)。

放松（5-10分钟）：静态拉伸，重点拉伸腹部和下背部。
平板支撑变式：前臂平板、直臂俯卧撑两种形式轮换，每次至少保持30秒以上，可逐渐过渡至单手或抬腿。
仰卧举腿：平躺，双腿伸直抬高至垂直于地面后缓慢放下，不让脚跟触地。此动作强化下腹部控制。
侧卧撑（Side Plank）：左右侧分别支撑15–30秒，保持肩膀、髋部和脚踝成一直线，锻炼侧腰和髋外侧肌群。
鸟狗式负重：四足姿势下做鸟狗式，可双手各持小哑铃或绑上脚踝沙袋，增加核心负荷。
扭转核心：坐姿两手空握或握单哑铃做俄罗斯转体，保持髋部稳固前倾，左右转体训练腹斜肌。
平板支撑及变式：标准平板支撑（肘或手撑地）是基础，可进阶单手或抬腿平板支撑，增强前链与核心整合。保持髋部不过分抬高或下沉。
鸟狗式（四足平衡）：四足跪姿一侧手臂和对侧腿同时伸直，保持骨盆稳定。可手臂向前伸时用哑铃轻抬以增加难度。
侧桥（侧平板支撑）：侧卧单肘支撑，另一手臂可伸直朝上或持哑铃向上举，锻炼侧腹肌、髋外侧肌群。两侧各做。
哑铃划船姿势：站姿屈膝前倾，双手下垂持哑铃于两侧，保持上背收紧。肩胛内收稳定，上拉哑铃至腰侧。可单侧支撑一腿保持平衡。
弹力带横向行走：弹力带套在双腿膝盖以上做侧移步行，加强髋外展肌群，提升下肢稳定性。

注意：每组动作强调慢速控制，下放和回位时发力均匀，避免依赖惯性。始终收紧腹部和臀部以支撑脊柱。

周三：心肺功能

目标：提升心肺耐力，改善整体健康。时长：45-60分钟
内容：

热身（5-10分钟）：动态拉伸 + 轻度有氧运动。
心肺训练（30-40分钟）：
慢跑或快走：中高强度，保持可对话的强度，20-30分钟 (Verywell Health Cardiorespiratory)。
弹力带开合跳：3组，每组10次，增加心率 (Medical News Today Cardio)。

放松（5-10分钟）：慢走或轻度拉伸。
原地快步跑：原地高抬腿或快速小步跑（跑动时脚步要快且高抬），锻炼心肺同时提高步频。
交叉步（侧滑步）：双腿微屈进行左右侧滑快速移动，保持低位稳定。
梯形脚步：利用假想梯形在地面用快速脚步完成“梯形”路线（前、后、左右），提高换向与敏捷度。
提膝跳（Knee tuck jumps）：连续跳起时双膝尽量向胸部提起，训练爆发力和快速抬腿能力。
Y冲刺停止：以Y字形（前中后或左右）方向进行短距离冲刺，每次冲刺后突然停住并换向。训练起动爆发和急停控制。

周四：连续弹跳与敏捷（初级）

目标：提高协调性和轻度爆发力，注重动作稳定性。时长：45-60分钟
内容：

热身（5-10分钟）：轻度有氧运动 + 动态拉伸。
弹跳训练（15-20分钟）：
下蹲跳（3秒保持）：下蹲后跳起，落地保持3秒，3组，每组8-10次 (Harvard Health Plyometrics)。
低台阶箱跳：用楼梯第一阶，双脚踏上，3组，每组8-10次 (ISSA Plyometrics)。
下蹲落：踮脚后快速下蹲，3组，每组10次 (HSS Plyometrics)。

敏捷训练（15-20分钟）：
高抬腿跑：原地高抬腿，注重协调，3组，每组1分钟。
弹力带侧向移动：用弹力带固定脚踝，侧向移动10步，每侧3组。

放松（5-10分钟）：静态拉伸，重点拉伸腿部和臀部。
原地深蹲跳：双脚与髋同宽站立，先做半蹲动作再向上蹬地跳起，落地时再次进入深蹲。注意落地膝盖微弯以吸收冲击，不要锁膝。
交替箭步跳：先做弓步深蹲（一个腿在前后），然后用力向上跳起交换双腿位置落地，训练下肢爆发力和协调。保证膝盖不要内扣。
跳箱/台阶跳：如果条件允许，可利用稳定的箱凳或踏步，双脚跳上跳下。初期高度低，集中注意膝关节对齐及膝盖稍屈以缓冲。
侧向跳跃：双脚并拢做左右交替小跳，扩展为单脚侧跳训练不同方向爆发。落地时控制身体重心。

注意：动作后脚跟先着地，再滚到前脚掌；始终保持膝盖与脚尖方向一致。如出现膝盖疼痛，应暂停并检查着地技术。训练量从少到多循序渐进，并随时关注动作质感。

周五：综合力量与稳定性

目标：增强整体力量和稳定性，支持功能性动作。时长：45-60分钟
内容：

热身（5-10分钟）：轻度有氧运动 + 动态拉伸。
力量训练（15-20分钟）：
哑铃高脚杯深蹲：用10kg哑铃，3组，每组12-15次。
哑铃单臂划船：3组，每组10-12次。
弹力带胸推：3组，每组12-15次。

稳定性训练（15-20分钟）：
单腿硬拉：用10kg哑铃，3组，每组10-12次。
侧卧支撑：3组，每侧20-30秒。

放松（5-10分钟）：静态拉伸，重点拉伸全身主要肌群。
深蹲（Squat）：双手各握哑铃垂于两肩或两侧，站立与髋同宽，下蹲至大腿平行地面位置后站起。锻炼大腿前侧、臀部和核心，注意膝盖不过脚尖。
箭步蹲（Lunge）：双手持哑铃站立，一腿向前跨出做深蹲（前腿膝盖弯曲，后腿膝盖接近地面），然后换腿。加强臀腿力量和平衡。
硬拉（Romanian Deadlift）：双手握哑铃放于身体前方，微屈膝，腰部前倾下放哑铃至胫骨中段，然后髋部用力拉回站立。主要锻炼臀部后链力量。
哑铃肩上推举：双手各持哑铃于肩膀高度，上半身直立或跪姿，向上推举哑铃至头顶。练习肩部力量与核心稳定。
哑铃划船（Bent-over Row）：站姿一膝一手支撑在凳上，另一手握哑铃从地面拉至腰部，增强背部肌群和肩胛控制。双侧交替进行。
农夫行走（Farmer’s Walk）：双手各握哑铃站立行走，保持躯干挺直不晃动，提升核心与握力。

注意：所有负重动作要保证脊柱中立、核心绷紧。初期可采用较轻负荷多组多次练习，感受动作模式；后期再提高重量或增加重复次数，应用渐进超负荷。

周六：休息或轻度活动

目标：促进恢复，保持轻度活动。时长：20-30分钟（可选）
内容：

20-30分钟慢走或瑜伽。
使用弹力带进行轻度筋膜松解（如滚压大腿）。

周日：休息

目标：完全休息，促进身体修复。
内容：避免剧烈运动，可进行轻度拉伸。
恢复策略

主动恢复：周三和周六进行轻度有氧运动，促进血液循环 (NASM Muscle Recovery)。
筋膜松解：高强度训练后（如周四、周五），用弹力带或瑜伽垫辅助滚压主要肌群（如股四头肌、臀部）5-10分钟。
静态拉伸：每次训练后进行5-10分钟静态拉伸，重点拉伸训练涉及的肌群。
睡眠：每晚7-9小时高质量睡眠，支持肌肉修复 (UCHealth Rest)。
营养：摄入充足蛋白质（每日1.6-2.2g/kg体重）和碳水化合物，保持水分充足。

日常运动与姿势调整

增加日常活动：
优先选择爬楼梯而非电梯。
1-3公里路程选择步行。
每天多次从站立到地面再站起（如做几个臀桥）。

姿势调整：
坐姿：保持脊柱中立，臀部坐骨承受体重，双腿平行，髋部内侧轻微发力 ([文章建议])。
站姿：双腿均匀分担体重，避免倚靠墙壁。
减少久坐：每30-60分钟起身活动1-2分钟。

动作意识：日常活动中（如洗碗、扫地）注意髋部稳定性和正确发力，避免腰部过度使用。
